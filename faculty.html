About Us

StudentAssist started as a static academic website and now works as a connected full-stack application.

Who we are

StudentAssist is an online learning platform that helps students discover courses, explore faculty, register for classes, and communicate with the institute through a real backend.

This version of the project uses Node.js for the backend and MySQL for data storage, which means the information shown on the website can now be created and retrieved dynamically.
